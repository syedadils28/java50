# Java50

A Java project repository containing various Java implementations and examples.

## Overview

This repository contains Java code examples, tutorials, and implementations covering fundamental to advanced Java concepts.

## Features

- Core Java concepts
- Object-oriented programming examples
- Data structures and algorithms
- Utility classes and helpers

## Getting Started

### Prerequisites

- Java 8 or higher
- Maven or Gradle (if applicable)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/syedadils28/java50.git
```

2. Navigate to the project directory:
```bash
cd java50
```

3. Build the project:
```bash
javac *.java
```

## Usage

Run Java programs with:
```bash
java ClassName
```
